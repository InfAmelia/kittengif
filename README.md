## Building urself

0. Get a mac
1. Install homebrew
2. Install ffmpeg

    brew install ffmpeg --with-libvpx --with-libvorbis --with-fdk-aacc

3. Edit/run `more-images.sh` to get the mp4s
4. Run `convert-to-webm.sh` to get the webm
